📊 Calculadora de Notas
Projeto simples desenvolvido para praticar HTML, CSS e JavaScript, permitindo calcular a média de notas de um aluno e exibir sua situação (Aprovado, Recuperação ou Reprovado).

🚀 Funcionalidades
Inserção de notas pelo usuário.
Cálculo automático da média.
Exibição do resultado na tela.
Indicação da situação do aluno:
✅ Aprovado
⚠️ Recuperação
❌ Reprovado

🎯 Objetivo do Projeto
Este projeto foi criado com fins educacionais para praticar:
- Manipulação do DOM.
- Captura de valores de formulários.
- Operações matemáticas com JavaScript.
- Estruturação de projetos Front-end.

🛠️ Tecnologias Utilizadas
HTML5 → Estrutura da página.
CSS3 → Estilização e layout.
JavaScript → Lógica de cálculo das notas.
